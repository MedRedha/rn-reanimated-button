# üöÄ Installation

Let's start by cloning the repo locally

```sh
git clone git@github.com:MedRedha/rn-reanimated-button.git
```

## üßº Clean-Up & Simple Run

Clean up the cache & install the dependencies

- `yarn clean`
- `yarn && yarn pod`
- `yarn start`
- `yarn ios` || `yarn android`
